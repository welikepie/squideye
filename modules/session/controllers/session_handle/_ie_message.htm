LemonStand is a modern web application that uses standards based technologies such as XHTML and JavaScript. Your web browser, Internet Explorer from Microsoft, unfortunately does not have adequate support for these technologies. Therefore, LemonStand cannot work in Internet Explorer at this time.

Fortunately, there are many web browsers available that do support web standards. These are available free of cost and are easy to get up and running. Here are 4 great choices that are available for both Windows and Mac:

  • FireFox
  • Google Chrome
  • Safari
  • Opera

After installing any one of these web browsers, you can open it up and attempt to access this page again. Using one of these web browsers will not only enable you to manage your website powered by LemonStand, but will also enhance your experience of many other websites.
