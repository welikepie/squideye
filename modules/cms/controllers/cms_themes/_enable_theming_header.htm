• Welcome to the Theming enabling tool. Please note that this operation is not undoable.

    To enable Theming you should perform the following steps.

    1. Create a new theme with the form below. All existing pages, partials and layouts will be moved to the new theme and it will be automatically activated.
    2. Copy resource files from your website resources directory (resources_dir_path ?>) to the theme resources directory (/themes/your-theme-code/resources) and update references to JavaScript, CSS and image files in your pages, partials and layouts as it is described in the documentation.
