The shop:payment_receipt action is designed for creating an order receipt page which you can display after a successful order payment. The action provides PHP variables required for displaying the order receipt and all information related to the order.

Variables

  • $order - an order object, loaded from the LemonStand database. An object of the Shop_Order class. The variable could be NULL if a requested order was not found in the database.
  • $payment_processed - indicated whether payment was already processed for the order. You can use this value to display a "Order not paid" message instead of the normal receipt page.
  • $items â€“ a list of order items. An object of the Db_DataCollection class. Each element of the collection is an object of the Shop_OrderItem class. This variable exists only if the requested order was found in the database.
