The shop:category action is designed for creating product category pages. The action loads a category object by a category URL name specified in the page URL.

Variables

  • $category - represents the category object (Shop_Category class). This variable can be NULL if a category, specified in the URL, was not found. Always check whether the $category variable is not NULL and display the "Category not found" message instead of the normal category page.
